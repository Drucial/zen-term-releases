# Third-party notices

ZenTerm is built on [Ghostty](https://ghostty.org) and ships the libraries,
fonts, and icons below. This file carries their copyright and license text,
which is what each of these licenses asks for.

Every library listed here was confirmed present in the shipped executable by
probing `Contents/MacOS/ZenTerm` itself, not by reading a dependency
manifest. What gets linked is a property of the build: a manifest lists what
could be linked, and `libghostty-fat.a` is an archive the linker draws from
selectively. Fonts are embedded data with no symbols, so they were found by
parsing sfnt table directories out of the same executable.

Ghostty pin at the time of this audit: `v1.3.1` (332b2ae). See
`docs/runbooks/third-party-notices.md` for how to re-probe when the pin moves.
